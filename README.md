📘 AI NoteMaster

텍스트 · 이미지 · PDF를 분석하여
요약 + 문제 생성 + 정답 제공까지 해주는 Android 학습 보조 앱

📱 앱 소개

AI NoteMaster는 사용자가 입력한 텍스트 또는 이미지(PDF/사진)를 분석하여
핵심 내용을 요약하고, 자동으로 문제와 정답을 생성해주는 Android 앱입니다.
학습 정리, 시험 대비, 복습에 활용할 수 있도록 설계되었습니다.

✨ 주요 기능

📝 텍스트 직접 입력 분석

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%ED%85%8D%EC%8A%A4%ED%8A%B8%20%EC%9E%85%EB%A0%A5.png" width="150" height="300"/>

텍스트를 직접 입력하여 ai에게 분석을 시킬 수 있습니다.(텍스트 직접입력은 한글, 영어 둘다 가능합니다.)

📷 사진 / PDF 업로드 후 분석

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EC%82%AC%EC%A7%84%20%EA%B6%8C%ED%95%9C.png " width="150" height="300"/><img src="https://github.com/ere-shark/AINoteMaster/blob/master/%ED%85%8D%EC%8A%A4%ED%8A%B8%20%EC%98%81%EC%96%B4.png" width="150" height="300]"/>

사진이나 PDF를 선택하면 OCR을 사용하여 영어를 텍스트 입력화면 텍스트 입력칸에 적어지며 분석하면 결과는 자동으로 한글로 출력됩니다.(사진과 PDF는 영어만 가능하며 사진을 선택할때는 권한을 허용해야합니다.)

🧠 AI 기반 노트 요약

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EB%B6%84%EC%84%9D%EA%B2%B0%EA%B3%BC%20%ED%95%9C%EA%B8%80.png" width="150" height="300"/>

텍스트 입력화면에서 분석하기 버튼을 누르면 AI가 분석해주고 결과화면에 요약문을 출력해줍니다.
❓❓ 자동 문제 생성

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EB%B6%84%EC%84%9D%EA%B2%B0%EA%B3%BC%20%ED%95%9C%EA%B8%80.png" width="150" height="300"/>

AI가 분석한 정보를 기반으로 문제를 생성해 요약문 아래칸에 출력해줍니다.(문제는 쉬움, 보통, 어렴움 난이도로 각 난이도당 3문제씩 생성해줍니다.)

✅ 정답 보기 기능

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EA%B2%B0%EA%B3%BC%20%EC%A0%95%EB%8B%B5png.png" width="150" height="300"/>

AI가 분석한 정보를 기반으로 생성한 문제의 답을 문제 출력칸 아래칸에 출력해줍니다.(정답은 정답보기 버튼을 눌러 확인할 수 있으며 정답숨기기 버튼으로 다시 가릴 수 있습니다.)

💾 분석 결과 저장 및 기록 관리

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EB%B6%84%EC%84%9D%EA%B2%B0%EA%B3%BC%20%ED%95%9C%EA%B8%80.png" width="150" height="300"/><img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EA%B8%B0%EB%A1%9D%ED%99%94%EB%A9%B4.png" width="150" height="300"/>

결과화면에 있는 저장하기 버튼을 누르면 제목을 정하여 저장할 수 있으며 저장한 정보는 기록화면에 제목이 목록으로 저장됩니다.(제목은 저장한 날짜를 순서로 목록에 나타납니다.)

📂 저장된 기록 상세 확인

<img src="https://github.com/ere-shark/AINoteMaster/blob/master/%EA%B8%B0%EB%A1%9D%EC%83%81%EC%84%B8.png" width="150" height="300"/>

기록화면에서 제목을 누르면 기록 상세화면으로 넘어가며 기록 상세화면에는 저장된 요약문, 문제, 정답이 있고 저장한 날짜까지 적혀있습니다.

🛠️ 사용 기술 (Tech Stack)

Language: Kotlin

IDE: Android Studio

UI: XML Layout

Database: Room

AI 처리: 텍스트 분석 기반 요약 및 문제 생성 로직

사진, PDF에서 텍스트 변환 및 분석 -> OCR

사용 AI API -) 내이버 Clova

기능: 이미지/PDF 입력 처리, 결과 저장 및 조회
